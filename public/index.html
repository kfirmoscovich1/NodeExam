Toys API

Method Route Description Example URLs
GET /toys Returns a list of toys
  • All toys: https://nodejs-mongodb-2ink.onrender.com//toys
  • Limit to 5 toys: https://nodejs-mongodb-2ink.onrender.com//toys?limit=5
  • Skip the first 10 toys: https://nodejs-mongodb-2ink.onrender.com//toys?limit=5&skip=10
GET /toys/search Search toys by name or info
  • Search for "robot": https://nodejs-mongodb-2ink.onrender.com//toys/search?s=robot
  • Search for "car": https://nodejs-mongodb-2ink.onrender.com//toys/search?s=car
  • Search for "block": https://nodejs-mongodb-2ink.onrender.com//toys/search?s=block
GET /toys/category Filter toys by category
  • Category "lego": https://nodejs-mongodb-2ink.onrender.com//toys/category?category=lego
  • Category "vehicles": https://nodejs-mongodb-2ink.onrender.com//toys/category?category=vehicles
  • Category "puzzle": https://nodejs-mongodb-2ink.onrender.com//toys/category?category=puzzle
GET /toys/prices Filter toys by price range
  • Price between 50 and 500: https://nodejs-mongodb-2ink.onrender.com//toys/prices?min=50&max=500
  • Price between 100 and 300: https://nodejs-mongodb-2ink.onrender.com//toys/prices?min=100&max=300
  • Price between 200 and 350 with skip: https://nodejs-mongodb-2ink.onrender.com//toys/prices?min=200&max=350&skip=4
GET /toys/single/:id Get a single toy by ID
  • Get toy with ID "6796707831153c4606e357a0": https://nodejs-mongodb-2ink.onrender.com//toys/single/6796707831153c4606e357a0
  • Get toy with ID "679670a031153c4606e357a2": https://nodejs-mongodb-2ink.onrender.com//toys/single/679670a031153c4606e357a2
POST /toys Create a new toy
*Token is required!*
  • Create a toy:
    {
    "name": "Toy Name",
    "info": "info",
    "category": "Action Figures",
    "img_url": "https://example.com/toy-image.jpg",
    "price": 1-999
    }
    https://nodejs-mongodb-2ink.onrender.com//toys
PUT /toys/{id} Update a toy by ID
*Token is required!*
  • Update toy with ID:
    {
    "name": "Toy Name",
    "info": "info",
    "category": "Action Figures",
    "img_url": "https://example.com/toy-image.jpg",
    "price": 1-999
    }
    https://nodejs-mongodb-2ink.onrender.com//toys/id
DELETE /toys/{id} Delete a toy by ID
*Token is required!*
  • Delete toy with ID:
    the ID of the toy to Delete
    https://nodejs-mongodb-2ink.onrender.com//toys/id

API Documentation

Toys API

Method Route Description Example URLs
GET /toys Returns a list of toys
  • All toys: http://localhost:3001/toys
  • Limit to 5 toys: http://localhost:3001/toys?limit=5
  • Skip the first 10 toys: http://localhost:3001/toys?limit=5&skip=10
GET /toys/search Search toys by name or info
  • Search for "robot": http://localhost:3001/toys/search?s=robot
  • Search for "car": http://localhost:3001/toys/search?s=car
  • Search for "block": http://localhost:3001/toys/search?s=block
GET /toys/category Filter toys by category
  • Category "lego": http://localhost:3001/toys/category?category=lego
  • Category "vehicles": http://localhost:3001/toys/category?category=vehicles
  • Category "puzzle": http://localhost:3001/toys/category?category=puzzle
GET /toys/prices Filter toys by price range
  • Price between 50 and 500: http://localhost:3001/toys/prices?min=50&max=500
  • Price between 100 and 300: http://localhost:3001/toys/prices?min=100&max=300
  • Price between 200 and 350 with skip: http://localhost:3001/toys/prices?min=200&max=350&skip=4
GET /toys/single/:id Get a single toy by ID
  • Get toy with ID "6796707831153c4606e357a0": http://localhost:3001/toys/single/6796707831153c4606e357a0
  • Get toy with ID "679670a031153c4606e357a2": http://localhost:3001/toys/single/679670a031153c4606e357a2
POST /toys Create a new toy
  • Create a toy: http://localhost:3001/toys
PUT /toys/{id} Update a toy by ID
  • Update toy with ID "6796704631153c4606e3579e": http://localhost:3001/toys/6796704631153c4606e3579e
DELETE /toys/{id} Delete a toy by ID
  • Delete toy with ID "6796700931153c4606e3579c": http://localhost:3001/toys/6796700931153c4606e3579c
